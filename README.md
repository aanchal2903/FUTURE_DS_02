# Facebook Ads Performance Dashboard – Task 2 (Future Interns)

## Overview
This project is part of **Task 2** from **Future Interns**: *Social Media Campaign Performance Tracker*.  
The goal is to analyze Facebook/Instagram ad campaign data and build an interactive Power BI dashboard to evaluate performance, engagement, CTR, and ROI.  

The dashboard provides actionable insights into campaign performance, top-performing posts, ROI analysis, and audience engagement.

---

## Dataset
- The project uses simulated/exported CSV data from Facebook Ads Manager.  
- Key columns include:  
  - Campaign ID  
  - Total Spend  
  - Total Clicks  
  - Total Impressions  
  - CTR (%)  
  - Approved Conversions  
  - Revenue  
  - Date  
  - Age, Gender, Interest  

> Note: The dataset was cleaned before analysis to handle missing values and errors in campaign IDs.  

---

## Key Features of the Dashboard
1. **KPI Overview** – Quick glance at Total Spend, Total Impressions, Total Clicks, CTR (%)  
2. **Campaign Engagement** – Bar chart showing total clicks by campaign  
3. **CTR (%) by Campaign** – Column chart visualizing click-through rates per campaign  
4. **Spend vs ROI Analysis** – Scatter chart mapping ROI against Total Spend  
5. **Performance Trends Over Time** – Dual-axis line chart showing Total Spend and Total Clicks  
6. **Click Performance & ROI Overview** – Donut / Pie charts showing campaign contribution and revenue vs spend  
7. **Interactive Filters** – Age, Gender, Campaign ID, Target Interest slicers for segmented analysis

## Insights & Recommendations
- Campaigns 936 and 916 show the **highest CTR and engagement**; consider scaling ad spend toward these campaigns.  
- Campaign 1178 has **lower ROI and CTR**; review targeting or creatives to improve performance.  
- Audience segmentation highlights **top-performing demographics** by Age, Gender, and Interest.  

---

## How to Use
1. Open the Power BI dashboard file (.pbix) in **Power BI Desktop**.  
2. Use the slicers to filter by **Age, Gender, Campaign ID, or Interest**.  
3. Hover over visuals for additional insights like CTR, ROI, and conversions.  
4. Explore trends, top campaigns, and ROI to make data-driven decisions.
